# Mutant DNA Analyzer

## service URL: http://3.229.94.148:5000

## Installation requirements:

```bash
NodeJS v12.14.1
```

## Install dependencies:

```bash
$ npm install
```
## How to run:

```bash
$ node app.js
```

## Tests

  To run the test suite (Jest), first install the dependencies, then run `npm test`:

```bash
$ npm install
$ npm test
```
